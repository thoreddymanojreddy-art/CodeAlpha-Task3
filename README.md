🌍 World Happiness Report Data Visualization Dashboard

CodeAlpha Data Analytics Internship – Task 3

This project focuses on creating meaningful and visually appealing data visualizations using the World Happiness Report dataset. The analysis explores happiness scores across countries and regions while examining factors such as GDP, social support, health, freedom, generosity, and corruption.

---

📌 Project Overview

Data visualization is a powerful tool for understanding patterns and trends in complex datasets. This project uses Python visualization libraries to analyze the World Happiness Report dataset and present insights through various charts and graphs.

The visualizations help identify relationships between happiness scores and socio-economic indicators while comparing countries and regions worldwide.

---

🎯 Project Objectives

- Visualize happiness scores across countries.
- Identify the happiest countries in the world.
- Compare average happiness levels across regions.
- Analyze the relationship between GDP and happiness.
- Study correlations among happiness factors.
- Examine regional distributions of happiness scores.
- Create professional and insightful visualizations.

---

📊 Dataset Information

The dataset is based on the World Happiness Report and contains country-wise happiness metrics.

Features Used

Feature| Description
Country| Country Name
Region| Geographic Region
Happiness Score| Overall Happiness Score
GDP per Capita| Economic Indicator
Social Support| Social Support Index
Health| Healthy Life Expectancy
Freedom| Freedom to Make Life Choices
Generosity| Generosity Score
Corruption| Perception of Corruption

🔑 Key Findings

🌍 Global Happiness Rankings

- Developed nations generally rank higher in happiness scores.
- Countries with strong economies and social support systems tend to perform better.

💰 GDP and Happiness

- A positive relationship exists between GDP per capita and happiness.
- Wealthier countries generally report higher life satisfaction.

🤝 Social Factors

- Social support and health strongly influence happiness scores.
- Freedom and generosity contribute positively to overall well-being.

🌎 Regional Insights

- Some regions consistently achieve higher happiness scores than others.
- Regional differences highlight economic and social disparities worldwide.

📊 Correlation Analysis

- GDP, health, and social support show strong positive correlations with happiness.
- Corruption generally exhibits a negative relationship with happiness levels.
